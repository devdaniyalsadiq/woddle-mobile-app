# Woddle App

Woddle is a cross-platform mobile application available for both Android and iOS. This README file provides instructions on how to set up and run the app on your local development environment.

---

## Prerequisites
Make sure you have the following installed:
- Node.js (version 18.19.1 or above)
- npm or Yarn (choose your preferred package manager)
- React Native CLI
- Android Studio (for Android development)
- Xcode (for iOS development)
- CocoaPods (for managing iOS dependencies)

---

## Getting Started

### Clone the Repository
